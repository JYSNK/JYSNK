Java数据类型

  1. 整数类型：包括byte（字节，8位）、short（短整型，16位）、int（整型，32位）和long（长整型，64位）。

  2. 浮点类型：包括float（单精度浮点型，32位）和double（双精度浮点型，64位）。

  3. 字符类型：char（字符类型，16位），用于表示单个Unicode字符。

  4. 布尔类型：boolean（布尔类型），用于表示真或假。

除了基本数据类型，Java还有引用数据类型，其中最常见的是类和接口。类是一种自定义的数据类型，可以包含属性（变量）和方法（函数）。接口是一种定义了一组抽象方法的类型，可以被类实现来提供具体的行为
